it has a menu with 6 options 
1.Add expense 
2.Change/set monthly goal (budget set)
3.View expenses 
4.Change custom percentage (your own custom percentage spent on when to remind you)
5.Clear data (clears the list) 
6.Exit
the value you put into the add expense will be saved in a .csv file
